Find an Energy Performance Certificate

Use this service to:

  • Find an EPC (Energy Performance Certificate) using the property's postcode.
  • Find an EPC using the 24-digit Report Reference Number printed on the report or provided by the Energy Assessor who produced it.
! Warning If the owner and/or occupier of the property has chosen to remove a property from general searches, the property information can only be found using the reference number on the certificate.
